
# Appbook

## Overview
This  is an application that deals with buying and selling books.Contains 2 categories of users:"user" and "admin" who is "user" but have privileges.

##Admin 
He can sell books.
He can do discounts in books 
He can Ban/UnBan users

##user
He can buy a book
He can search a book
He can Read a book
He can do a review in  a book
He take notifications for every admin action.
